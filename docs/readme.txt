Para correr o notebook é necessário: 
    -Fazer download do dataset fornecido (https://www.kaggle.com/hugomathien/soccer)
    -Fazer do dataset suplementar (https://www.kaggle.com/jiezi2004/soccer), apenas são necessários os ficheiros:
        -shotoff_detail.csv
        -shoton_detail.csv
        -possession_detail.csv
        -corner_detail.csv
        -cross_detail.csv
    -Copiar os ficheiros para uma subpasta que deve estar na pasta do notebook com o nome 'datasets'
    -É também necessário instalar algumas bibliotecas adicionais que contêm algoritmos que foram usados:
        -numpy
        -pandas
        -matplotlib
        -sklearn
        -xgboost
        -keras
    -Após estes passos é possível iniciar e correr o jupyter notebook.
